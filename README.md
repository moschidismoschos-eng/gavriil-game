# Το Συνεργείο του Μικρού Μηχανικού (Παιδικό παιχνίδι)

Λειτουργεί σε browser (κινητό/PC) με HTML/CSS/JS.

## Τι έχει μέσα
- 15 τύποι οχημάτων
- 25 πίστες (levels) με αυξανόμενες βλάβες
- Endless mode (μετά την πίστα 25)
- Διάγνωση + εργαλειοθήκη
- Μίνι παιχνίδι αλλαγής λάστιχου (σειρά μπουλονιών)
- Πόντοι/αστέρια ανάλογα με χρόνο

## Πώς το τρέχεις (από Termux)
Μπες στον φάκελο και τρέξε:

python -m http.server 8000

Μετά άνοιξε:
http://localhost:8000
